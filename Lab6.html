Висновки

У результаті виконання проєкту було створено інтерактивний застосунок для пошуку зображень із використанням публічного REST API сервісу Pixabay. Під час розробки здобуто практичні навички роботи з асинхронними операціями в JavaScript, зокрема з використанням промісів та методів then(), catch(), finally(), а також з ланцюжками промісів. Ознайомлено з принципами роботи HTTP/HTTPS протоколів, особливостями формування запитів та отримання відповідей. Вивчено формат REST API, роль HTTP-методів та заголовків у комунікації з сервером. Реалізовано асинхронну взаємодію з сервером за допомогою Fetch API та обробку результатів без перезавантаження сторінки (AJAX). Також отримано досвід роботи з крос-доменними запитами, обробкою помилок, підключенням сторонніх бібліотек для покращення UI (iziToast, SimpleLightbox), і оптимізацією відображення контенту. Проєкт сприяв кращому розумінню сучасного підходу до побудови веб-застосунків з використанням REST API.
